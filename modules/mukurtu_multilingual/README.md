# Description

This module provides default configuration to jumpstart a multilingual website
on Mukurtu CMS. It enables and configures the necessary Drupal core multilingual
modules to support content, config and interface translation.

## Core Dependencies

This module depends on and enables:
* Interface Translation (`locale`) - Core module for managing languages and
translations.
* Content Translation (`content_translation`) - Enables translation of content
entities.
* Configuration Translation (`config_translation`) - Enables translation of
configuration (menus, views, etc.).

## Usage

Enable this module to jumpstart configuration of multilingual support for your
Mukurtu site. Once enabled, you can add additional languages and begin
translating content and configuration through Drupal's standard translation
tooling.
